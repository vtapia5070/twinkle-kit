## Usage

### Light Theme

- **Background**: `#ffffff` (neutral-1)
- **Surface/Cards**: `#f9f9fb` (neutral-2)
- **Primary/Buttons**: `#943cdd` (primary-9)
- **Text**: `#1a1a1a` (neutral-11)

### Dark Theme

- **Background**: `#111111` (neutral-1)
- **Surface/Cards**: `#19191b` (neutral-2)
- **Primary/Buttons**: `#943cdd` (primary-9)
- **Text**: `#f5f5f5` (neutral-11)

### Button States

- **Normal**: `bg-primary-9` with `text-primary-contrast`
- **Hover**: `bg-primary-10`
- **Disabled**: `bg-primary-disabled` (#797b86)

### Alert Types

- **Info**: `bg-primary-3` (#f6edff) with `text-primary-11` (#8632ca)
- **Error/Success/Warning**: Colored background with white text
