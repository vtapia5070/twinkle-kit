## Color Meanings

### Primary Colors

- **primary-1**: Lightest purple for subtle backgrounds
- **primary-3**: Light purple for alert backgrounds
- **primary-8**: Focus states and interactive elements
- **primary-9**: Main brand/accent color
- **primary-10**: Hover states
- **primary-11**: Accent text color

### Neutral Colors

- **neutral-1**: Page background
- **neutral-2**: Card/surface background
- **neutral-3**: Disabled states, subtle borders
- **neutral-8**: Secondary text, icons
- **neutral-9**: Disabled button backgrounds
- **neutral-11**: Primary text color

### Semantic Mappings

#### Light Theme

- **Background**: `neutral-1` (`#ffffff`)
- **Surface**: `neutral-2` (`#f9f9fb`)
- **Text**: `neutral-11` (`#1a1a1a`)
- **Secondary Text**: `neutral-8` (`#8b8d98`)
- **Primary/Accent**: `primary-9` (`#943cdd`)

#### Dark Theme

- **Background**: `neutral-1` (`#111111`)
- **Surface**: `neutral-2` (`#19191b`)
- **Text**: `neutral-11` (`#f5f5f5`)
- **Secondary Text**: `neutral-8` (`#8b8d98`)
- **Primary/Accent**: `primary-9` (`#943cdd`)

## CSS Variables

All colors are available as CSS custom properties:

```css
/* Primary colors */
--color-primary-1, --color-primary-3, --color-primary-8
--color-primary-9, --color-primary-10, --color-primary-11

/* Neutral colors */
--color-neutral-1, --color-neutral-2, --color-neutral-3
--color-neutral-8, --color-neutral-9, --color-neutral-11

/* Semantic colors */
--color-background, --color-surface, --color-surface-hover
--color-border, --color-text-primary, --color-text-secondary
--color-text-disabled, --color-primary, --color-primary-hover
--color-primary-active, --color-primary-disabled
--color-primary-contrast

/* Alert colors */
--color-alert-info-bg, --color-alert-info-text

/* State colors */
--color-error, --color-success, --color-warning
```

## Important Distinction

- **`--color-text-primary`** = Readable text color (optimized for accessibility)
- **`--color-primary`** = Brand primary color (for accents and highlights)

This ensures proper contrast while providing brand expression options.
